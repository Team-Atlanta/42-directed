<task type="auto">
  <name>Task 3: Create parse_diff.py for function detection</name>
  <files>oss-crs/scripts/parse_diff.py</files>
  <action>
Create parse_diff.py that parses unified diff files and detects affected functions using tree-sitter:

1. Create oss-crs/scripts/ directory if not exists

2. Implement parse_diff.py following the research pattern:
   ```python
   #!/usr/bin/env python3
   """Parse unified diff files and identify affected functions using tree-sitter.

   Output format: path function_name (one per line)
   This matches the slice target file format expected by the LLVM analyzer.
   """

   import os
   import sys
   from pathlib import Path

   from unidiff import PatchSet
   import tree_sitter_c as tsc
   from tree_sitter import Language, Parser
   ```

3. Implement parse_diff_to_functions(diff_dir, src_root) function:
   - Iterate over all .diff/.patch files in diff_dir
   - Parse each with unidiff.PatchSet
   - For each changed file, identify line numbers
   - Parse source file with tree-sitter to find function_definition nodes
   - Check if any changed line falls within function boundaries
   - Output "path function_name" for each match

4. Handle edge cases:
   - Skip non-C/C++ files (check extensions: .c, .cc, .cpp, .cxx, .h, .hpp)
   - Skip if source file doesn't exist (deleted files)
   - Handle both added and modified files
   - For C++, use tree-sitter-cpp if available, fallback to C parser

5. Main entry point:
   ```python
   if __name__ == "__main__":
       if len(sys.argv) != 3:
           print("Usage: parse_diff.py <diff_dir> <src_root>", file=sys.stderr)
           sys.exit(1)

       diff_dir = sys.argv[1]
       src_root = sys.argv[2]

       functions = parse_diff_to_functions(diff_dir, src_root)
       for path, func_name in functions:
           print(f"{path} {func_name}")
   ```

6. Make script executable with shebang
